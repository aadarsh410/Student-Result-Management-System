# 🎓 Student Result Management System

A desktop-based **Student Result Management System** developed using **Python, Tkinter, SQLite, and Pillow (PIL)**. The application simplifies student record management, course administration, and examination result processing through an intuitive graphical user interface.

---

## 📌 Overview

This project was developed to automate the process of managing student information and examination results. It provides administrators with a secure platform to manage students, courses, and academic performance efficiently.

---

## ✨ Features

- 🔐 Secure Login & Registration
- 🔑 SHA-256 Password Hashing
- 👨‍🎓 Student Management
- 📚 Course Management
- 📝 Result Management
- 📊 Automatic Percentage Calculation
- 🔍 Search Student & Course Records
- 📋 View & Delete Results
- 💾 SQLite Database Integration
- 🖥️ User-Friendly GUI using Tkinter

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/aadarsh410/Student-Result-Management-System.git
```

### Move into Project

```bash
cd Student-Result-Management-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python login.py
```
